# 架构决策记录（ADR）

## 记录模板

### ADR-001：决策标题

- 状态：已采纳 / 已废弃 / 讨论中
- 日期：YYYY-MM-DD
- 背景：为什么需要这个决策
- 决策：最终选择了什么方案
- 备选方案：考虑过哪些其他方案
- 影响：会对代码、流程、协作带来什么变化
